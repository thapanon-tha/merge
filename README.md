# Merge Test

This project is TypeScript project to Merge number between collection.

- [Installation](#installation)

## Installation
1) Clone the repository

   ```shell
   git clone https://github.com/thapanon-tha/merge.git
   ```
2) Goto folder
   ```bash
   cd ./merge
   ```
3) Install package
   ```bash
   npm install
   ```

3) Start Project
   ```bash
   npm start
   ```
   or
    ```bash
   npm run start
   ```
3) Test Project
   ```bash
   npm run test
   ```
